# Visualization Tools

## Add these files:

```
msa_visualizer.py    ← MSA alignment visualization with MAFFT
csv2msa.py           ← Convert CSV output to MSA format
```

## From your KA-Search directory:

```bash
cp ~/KA-Search/align_vs_lead_*.py ./msa_visualizer.py
cp ~/KA-Search/csv2msa.py ./csv2msa.py
```

These tools create publication-quality MSA visualizations of designer output.
